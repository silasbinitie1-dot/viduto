# Viduto App

A modern video creation platform built with React and Vite.

## Features

- AI-powered video creation
- Product showcase videos
- Text-based video generation
- Modern, responsive design

## Getting Started

```bash
npm install
npm run dev
```

## Building the app

```bash
npm run build
```

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- React Router
- Radix UI components
